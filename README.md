#  Personal Assistant using Python

This project is a voice-activated Personal Assistant built using Python. It can recognize speech, process commands, and perform various tasks such as searching the web, telling the time, opening applications, and more.

#  Features

Speech recognition
Text-to-speech response
Open applications (YouTube, Google, etc.)
Search information from the web
Tell date & time
Play music
Basic conversational responses

#  Tech Stack

Python
SpeechRecognition
pyttsx3
Webbrowser
datetime
Wikipedia API

#  Project Structure

/personal-assistant
  assistant.py
  requirements.txt
  README.md

# How to Run

pip install -r requirements.txt
python assistant.py
